# publishing lib
* package.json
  ```
    files": [
        "/lib"
    ]
  ```
* create account https://www.npmjs.com/
* npm login
* npm publish
  
# update version and publishing lib
* git add/commit 
* npm version <new_Version_No>
* npm publish
